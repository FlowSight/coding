# Time Copilot — Overview

## What
- LLM-based copilot for time entry creation, submission, and approval
- Interactive — requires human instruction and interaction
- Part of Dynamics 365 Project Operations Copilot (alongside Project Copilot)

## Why
- Reduce friction in timesheet workflows
- Help users who forget or delay time entry submission
- High visibility: CVP-level feature

## How
- LLM-based with copilot orchestration framework
- Shared framework with Project Copilot for LLM parameters and model deployments
- User provides instruction → Copilot executes
