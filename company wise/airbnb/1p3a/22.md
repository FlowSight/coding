Given a list of menu items with their prices and several choice of menu bundles, select the combination that comes closest to the budget. You can choose up to 3 menu bundles. Write a function to find the best bundle combination. The total price of bundles should not exceed the given budget. For example, given the following input:

Menu items and prices:
A: $10
B: $20
C: $30
Bundle choices:
Bundle1 (A + B): $25
Bundle2 (B + C): $45
Bundle3 (A + B + C): $55 Given a budget, find the best combination.
Provide at least 5 test cases. The function signature is def choose_menu_bundles(menus, budget):.
