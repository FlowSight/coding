Given a series of jobs, each job has a start time, an end time, and a profit. You need to choose some jobs that do not overlap and can achieve the maximum total profit.

Input
Each job is represented by a list of three integers: [start time, end time, profit].
The job list is a 2D list with details of n jobs.
Output
Output the maximum profit that can be obtained.
Example
Input: jobs = [[1, 3, 50], [6, 19, 100], [2, 100, 200]] Output: 250

Constraints
1 <= n <= 10^4
1 <= jobs[i][0] < jobs[i][1] <= 10^9
1 <= jobs[i][2] <= 10^4
