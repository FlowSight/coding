Design and implement a simple wallet system for Airbnb. The wallet should support the following functionalities:

Users can create a wallet.
Users can deposit and withdraw funds.
Users can check the balance.
While implementing:

All functions should be class methods.
Concurrency does not need to be handled.
Each user can have only one wallet.
Test cases:

Create a wallet.
Deposit 100 units of funds into the wallet.
Checking the balance should return 100.
Withdraw 50 units of funds from the wallet.
Checking the balance should return 50.
Attempt to withdraw 60 units of funds, which should fail and keep the balance as 50.
Data Scale:

Assume support for up to 1000 users.
Single wallet funds do not exceed 10000 units.
