Given a set of tasks, each with a deadline and a profit, schedule tasks to maximize total profit. Each task takes one unit of time. If multiple tasks can be performed at the same time unit, prefer the ones with higher profit. Write a function that takes in a list of tasks and outputs the schedule with the maximum profit.

Input:

A list of tasks, where each task is represented as (profit, deadline).
Output:

The maximum possible profit.
Assumptions:

The number of tasks does not exceed 1000.
Each task's profit is a non-negative integer.
Example
Input: [(100, 2), (19, 1), (27, 2), (25, 1), (15, 3)] Output: 142

Hint
Use a greedy algorithm starting with the task with the maximum profit.
Optimize the search for free slots.
