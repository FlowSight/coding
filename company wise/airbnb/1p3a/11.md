Sugget a split staty where you stay at 2 different houses, one for the first p[art of your trip and other for the remainder. Mkae an api endpopint that will return all possible combo of split stays across 2 listings for a given date range. Give each listing a name ("A" "B", etc) and our availability will just be a list of day numbers.

For example: A - [1,2,3,6,7,10,11]; B - [3,4,5,6,8,9,10,13]; C - [7,8,9,10,11]
Given the date range [3-11], the expected result is all sets of two Airbnbs that could form a split stay: [B, C]
暴力解法，加分项写出了bit mask解法
参考暴力解法
from itertools import combinations

def find_splits(listings, start_date, end_date):
desired_days = set(range(start_date, enddate+1))
splits = []
for a,b in combinations(listings.keys(), 2):
a_set = set(listings[a])
b_set = set(listings)
if desired_days.issubset(a_set) or desired_days.issubset(b_set):
continue
combined_days = set(listings[a]) | set(listings)
if desired_days.issubset(combined_days):
splits.append((a,b))
for a in listings.keys():
a_set = set(listings[a])
if desired_days.issubset(a_set):
splits.append((a))
return splits

listings = {"A": [1,2,3,6,7,10,11], "B":[3,4,5,6,8,9,10,13], "C":[7,8,9,10,11]}
print(find_splits(listings, 3,11))
