Design and implement a class that supports the following methods:

set_value(key: str, value: int): Set the value of a specified key.
set_sum(key: str, values: List[str]): Set the value of a specified key to the sum of multiple keys' values.
get_value(key: str): Get the current value of a specified key.
The class should efficiently support these operations, with the follow up being to achieve average O(1) time complexity for all operations.

Example Cases:

```
obj = YourClass()
obj.set_value("a", 5)
obj.set_sum("b", ["a", "a"])
print(obj.get_value("a"))  # Expected output: 5
print(obj.get_value("b"))  # Expected output: 10
```
Data Scale:

These methods should attain average O(1) time complexity.
