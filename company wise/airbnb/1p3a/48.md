Given a Directed Acyclic Graph (DAG), nodes represent checkpoints in the skiing process, each node v has a score w(v); an edge (u→v) represents a path from node u to node v, with a time cost (weight) t(u,v).

The start point is 'start', with a score w(start)=0, and the terminal nodes begin with an underscore '_'.

The goal is to find the path from 'start' to any terminal node that maximizes the path score, defined as:

Score(P) = sum of scores of nodes in path P - sum of time costs of edges in path P

Input:

Number of nodes n, number of edges m
n lines containing nodes and their scores (string, integer)
m lines containing edge information (start node, end node, time cost)
Output: The maximum path score

Example Input:

5 6
start 0
A 1
B 2
_C 3
_D 4
start A 2
A B 3
A _C 5
B _D 6
C _D 2
D B 1
Example Output:

Maximum path score
