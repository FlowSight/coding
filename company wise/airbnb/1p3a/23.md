In a task scheduling system, we have a series of tasks represented by specific characters, and each task must have at least n cooling time units between them. Write a function to schedule these tasks to be completed in the shortest time possible. Assume the CPU can be on standby at any time. Given a list of tasks and the cooling time, return the shortest completion time of the tasks.

Example
Input: tasks = ['A', 'A', 'A', 'B', 'B', 'B'], n = 2
Output: 8
Explanation: A -> B -> idle -> A -> B -> idle -> A -> B
Constraints
The number of tasks is between [1, 10000].
The task cooling time is between [0, 100].

Example
Input: set('apple', 3)
Input: get('apple')
Output: 3
Input: set('app', 5)
Input: getSum('ap')
Output: 8
Constraints
All keys are combinations of lowercase letters with a length of [1, 100].
