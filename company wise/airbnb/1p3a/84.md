The backend senior-level exam consisted of one round of coding, one round of design, one round of project presentation, and one round of code review.
The code was a backtracking problem, but the description borrowed some concepts from renting apartments. The gist was that a group of people were going on a trip and needed to be housed in the same area. The goal was to minimize the number of rooms used, given that each room had a capacity and location. The solution was to first iterate through the list of rooms in the target area, then backtrack through all rooms to find the optimal solution. Essentially, it was a problem of finding the minimum number of positive integers in an array to make a number greater than the target number.

A tourist group needs to rent rooms in the same area. Each room has a capacity and a location area. Design an algorithm to find the combination of rooms with the minimum number of rooms such that all people can be accommodated. Assume there are n rooms, each with the following constraints:

Capacity: vi
Location area: ai
Input:

Integer array rooms [] representing the capacity of each room
Output:

Minimum number of rooms needed
Example
Input: rooms = [(2, 'A'), (4, 'A'), (1, 'A'), (6, 'B')], groupsize = 5

Output: 2

Constraints
1 <= n <= 20
1 <= vi <= 10
groupsize <= 100
