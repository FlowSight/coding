Given an m x n board. Each cell is a string like G1, W0, S2:

The first character T is the terrain type.
The following digit C (0–9) is the number of crowns in that cell.
Connected component: cells with the same terrain type connected by 4-directional adjacency (up/down/left/right).

Component score:

score = (number of cells in the component) × (total crowns in the component)

Return the total score as the sum of all component scores.

Input (stdin)
Line 1: m n
Next m lines: n strings separated by spaces
Output (stdout)
One integer: total score
Constraints
1 <= m, n <= 200
crowns per cell is 0..9

G1 G2 W0 W1 S1
G2 G3 W0 W1 S1
