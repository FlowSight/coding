Given a string of payment data [timestamp, value, id],
you were required to write APIs for insert, get, and getValueInRange.


Design and implement a class to manage payment data with the following operations:

insert(timestamp: int, value: float, id: str): Insert payment data consisting of a timestamp, payment amount, and an identifier.

get(id: str) -> (timestamp: int, value: float): Get the timestamp and payment amount by the unique identifier of the payment data.

getValueInRange(start: int, end: int) -> List[float]: Return a list of all payment values within the given timestamp range [start, end].

Handle edge cases and boundary conditions appropriately. Assume timestamps are strictly monotonically increasing.

Input/Output Format
insert: No return value.
get: Return a tuple (timestamp, value).
getValueInRange: Return a list of floating-point numbers.
Example
Assuming the following sequence of insertions:

insert(1, 100.0, 'a')
insert(2, 200.0, 'b')
insert(3, 300.0, 'c')
Calling get('b') returns (2, 200.0).

Calling getValueInRange(1, 2) returns [100.0, 200.0].
