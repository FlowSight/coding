Given a list of integers, return the smallest number that can be formed by concatenating the integers. The input is a list of integers, and the output is their concatenated smallest number string.

Input
A list of integers where 0 <= list length <= 100, and each integer is a digit from 0 to 9.
Output
A string representing the smallest number that can be formed.
Example
Input: [7,1,8] Output: "178"
Input: [0,4,0] Output: "4"



Given a list of integers and a lower bound integer, return the smallest number that can be formed by concatenating the integers that is greater than the lower bound. The input is a list of integers and a lower bound integer, and the output is the smallest number string greater than the lower bound.

Input
A list of integers where 0 <= list length <= 100, and each integer is a digit from 0 to 9.
An integer representing the lower bound where 0 <= lower bound <= 10^9.
Output
A string representing the smallest number that can be formed that is greater than the lower bound.
Example
Input: [7,1,8], lower=200 Output: "718"
Input: [7,1,8], lower=179 Output: "187"
