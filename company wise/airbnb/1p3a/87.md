Given an integer array input_array, implement a function encode_rle(input_array) to calculate the Run-Length Encoding (RLE) of this array.

Input: An integer array, e.g., (1, 1, 1, 3, 6, 6, 1, 1).

Output: An RLE representation of the array, e.g., ((3, 1), (1, 3), (2, 6), (2, 1)); this means that there are 3 occurrences of 1, 1 occurrence of 3, 2 occurrences of 6, and 2 occurrences of 1 again.

Implement the corresponding encoding and decoding functions, and verify the functionality through actual encoding and decoding under the given inputs. Provide the following test cases and data scale to ensure that the input array can be accurately restored:

Test Cases:

Input (1, 1, 1, 3, 6, 6, 1, 1) Output ((3, 1), (1, 3), (2, 6), (2, 1))
Input (1) Output ((1, 1))
Data scale: The length of input_array can reach 10^5 and the value of elements is between [-10^9, 10^9].
