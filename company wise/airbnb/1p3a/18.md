Design a multi-stream iterator. The input is multiple character streams that provide byte data. The iterator should read bytes sequentially from each stream until all streams are exhausted.

Requirements
Implement a class MultiStreamIterator, whose constructor accepts a list of character streams.
Provide a method has_next() that returns whether there are more bytes to read.
Provide a method next() that returns the next byte of data.
Example
Input:
Stream1: "a1b"
Stream2: "c2"
Stream3: "3"
Output:
"a", "c", "3", "1", "2", "b"
Constraints
Each stream contains at most 1000 characters.
Characters are ASCII characters.
