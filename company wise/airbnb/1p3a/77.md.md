Given a set of tasks, each with a deadline and profit. You need to complete the tasks before the deadlines to earn the profit. Each task requires only one day, and you can complete only one task per day. Write a function to compute the maximum total profit and the order of tasks to be completed.

Input: A list of tuples, each containing a task name, deadline, and profit, e.g., [("task1", 1, 20), ("task2", 2, 15), ("task3", 2, 10)].

Output: A tuple containing the list of task execution order and the maximum profit, e.g., (["task1", "task2"], 35).

Test cases:

Input: [("task1", 2, 50), ("task2", 1, 20), ("task3", 2, 60)] Output: (["task3", "task1"], 110)

Input: [("task1", 1, 10), ("task2", 1, 20), ("task3", 1, 30)] Output: (["task3"], 30)

Constraints: Up to 1000 tasks, maximum deadline of 1000.
