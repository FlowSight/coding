Search

Given an array of strings, for each string, find the shortest substring that it maps to uniquely and not any other string in the array.

Example
Input: ['babab', 'ababc', 'abc']

Output: ['bab', 'ab', 'c']

Note: The order of output must be consistent with the input array.

Constraints
The length of the array is n, where 1 <= n <= 100.
The length of each string is m, where 1 <= m <= 100.
All strings are made of lowercase letters.
