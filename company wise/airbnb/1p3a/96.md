Problem Description
Implement the classic Connect 4 game. In this game, players take turns dropping chips into a grid from the top. The game board consists of 6 rows and 7 columns. After a player's move, the game engine should check for four of the same color chips connected horizontally, vertically, or diagonally, in which case the game ends with that player's victory.

Task
Create a function connect4(moves).

moves is a list of integers each representing a column where a chip is dropped (0-indexed).

Return the winning player's mark (1 or 2). If the game ends without a player winning, return 0.

Example
Input: moves = [0, 0, 1, 1, 2, 2, 3]
Output: 1
Constraints
The board is 6 rows by 7 columns.
Each player places one chip in any non-full column per turn.
There will be no more than 42 moves in total.
The input is guaranteed to be valid, and the game ends only when four consecutive chips appear.
