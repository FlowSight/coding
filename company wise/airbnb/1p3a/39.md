Given an auto-generated four-digit number, such as 2345, and a built-in guesser function that, for each input numeric string, returns the count of digits matching in the correct position against the generated number. Use this mechanism to write a method that accurately guesses all the random four-digit numbers. Example Input:

Input: 2222, Output: 1

Only 2 is in the correct position.
Input: 1341, Output: 2

Digits 3 and 4 are in the correct position.
Requirement: Design a method to automatically guess the four-digit number until the correct number is guessed. Provide test cases to verify the correctness of your solution.
