Implement a program to determine the result of a Connect 4 game. Given a 6x7 board array containing 'R' (red player), 'Y' (yellow player), and spaces, determine which player has won the game, if the game has ended in a draw, or if the game is still ongoing.

Input
A 6x7 2D character array representing the game board.
Output
Return 'R' if the red player wins.
Return 'Y' if the yellow player wins.
Return 'D' if the game ends in a draw.
Return 'N' if the game is still ongoing.
Example
Input:
[[' ', ' ', ' ', ' ', ' ', ' ', ' '],
 [' ', ' ', ' ', ' ', ' ', ' ', ' '],
 [' ', ' ', ' ', 'R', ' ', ' ', ' '],
 [' ', ' ', 'Y', 'R', ' ', ' ', ' '],
 [' ', 'Y', 'Y', 'R', 'R', 'Y', ' '],
 ['R', 'R', 'Y', 'R', 'Y', 'Y', 'R']]
Output: 'R'
Constraints
The 2D array is always of size 6x7.
