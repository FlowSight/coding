Given a URL containing query parameters, parse and return a dictionary containing all key-value pairs from the URL.

Input Description:

The input consists of a single string representing a URL.
Output Description:

Return a dictionary containing all the query parameters as key-value pairs. For keys without values, set the value to true. For keys with multiple values, the value should be a list. Handle URL-encoded characters, e.g., %26 should be replaced with &, and %3D should be replaced with =.
Examples:

Input: http://www/api/v1/movie?year=100&type=action Output: {'year': 100, 'type': 'action'}

Input: http://www/api/v1/movie?book Output: {'book': true}

Input: http://www/api/v1/movie?type=scifi&type=action Output: {'type': ['scifi', 'action']}

Input: http://www/api/v1/movie?type%26view=scifi&area%3Dcity=action Output: {'type&view': 'scifi', 'area=city': 'action'}
