Design a data structure that supports adding a number from a data stream and finding the median of all numbers in the stream in O(1) time complexity. Implement a class with two methods, addNum(int num) which adds an integer to the data stream, and findMedian() which returns the median of all elements added so far.

Parameters

addNum(int num): Adds the integer num to the stream.
findMedian() -> float: Returns the median of all numbers added. If the count of numbers is even, return the average of the two middle numbers.
Example

addNum(1)
addNum(2)
findMedian() # Returns 1.5
addNum(3) 
findMedian() # Returns 2
Constraints

number of operations: up to 105
value range: 0 <= num <= 106
Your implementation should consider time and space efficiency.
