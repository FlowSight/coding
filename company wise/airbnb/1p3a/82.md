Given a vector of strings where each string contains several segments separated by spaces, with each segment consisting of a letter and a score. The task is to calculate the total score for each letter. If a letter appears multiple times, the total score is the sum of scores for that letter multiplied by the number of times it appears. Implement a function that takes the vector of strings as input and returns the total score for each letter. Example input: {"f1 g2 t2", "d2 f2 t1"}, example output: {'f': 6, 'g': 2, 't': 4, 'd': 2}. Assume all letters are lowercase and scores are positive integers.
