Given the number of courses and a list of their prerequisites, determine if it's possible to finish all courses. Courses are numbered from 0 to n - 1.

For example, given numCourses = 2 and prerequisites = [[1, 0]], it means course 1 depends on course 0 and should return true as they can be completed in sequence.

Input:

numCourses: Total number of courses (0 <= numCourses <= 10^5)
prerequisites: List of prerequisite pairs
Output:

Whether it is possible to complete all courses
Example:

Input: numCourses = 2, prerequisites = [[1, 0]]
Output: true
Input: numCourses = 2, prerequisites = [[1, 0], [0, 1]]
Output: false
Constraints:

0 <= numCourses <= 10^5
