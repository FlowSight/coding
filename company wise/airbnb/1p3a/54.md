Assume we have a list of Properties, each defined by id, neighborhood, and capacity. Given a target Neighborhood and GroupSize, find a recommended combination of property ids within the specified neighborhood that has the least number of properties with a total capacity greater than the groupSize and as close as possible to it. If no such combination exists, return an empty array. Implement an algorithm to solve this.

Input:

Properties: A list of Property objects, each with attributes: id (integer), neighborhood (string), capacity (integer).
Neighborhood: A string representing the target neighborhood.
GroupSize: An integer representing the target group size.
Output:

Return an array containing the recommended property ids. If no suitable combination is found, return an empty array.

Example:

Properties = [Property(1, "downtown", 5), Property(2, "downtown", 3), Property(3, "downtown", 1), Property(4, "uptown", 4), Property(5, "uptown", 2)]
Neighborhood = "downtown"
GroupSize = 4  -> {1}
GroupSize = 10 -> {1, 3, 4}/{1, 2, 5}
GroupSize = 6 -> {1, 3}
GroupSize = 3 -> {2}
GroupSize = 16 -> {}
