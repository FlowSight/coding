Given an integer array nums, find the contiguous subarray (containing at least one number) which has the largest product and return its product.

Input
An integer array nums, with length in the range [1, 2 * 10^4].
Each element's value in the range [-10, 10].
Output
An integer which is the largest product.
Example
Input: [2,3,-2,4]

Output: 6

Input: [-2,0,-1]

Output: 0
