Implement a function that receives an array of integers and a target floating-point number, and returns all possible combinations of numbers that sum up to the target number. Each number may be used unlimited times. Each combination in the result should be in non-descending order, and the result set must not contain duplicate combinations.

Input:

nums (List[float]): Array of integers and floats, representing candidate numbers.
target (float): Target floating-point number.
Output:

List[List[float]]: List of combinations that meet the criteria.
Example:

nums = [2.5, 3.0, 6.5, 7.0]
target = 7.5
Output:

[[2.5, 2.5, 2.5], [3.0, 3.0, 1.5], [7.0]]
