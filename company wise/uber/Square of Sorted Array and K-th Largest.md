Given an integer array nums sorted in non-decreasing order, return an array of the squares of each number sorted in non-decreasing order. Then, find the k-th largest element in the sorted array.

Input format:

Integer array nums and integer k
Output format:

Integer, the k-th largest squared number
Example 1:

Input: nums = [-4,-1,0,3,10], k = 3
Output: 9
Constraints:

1 <= nums.length <= 10000
-10000 <= nums[i] <= 10000
1 <= k <= nums.length
Example
Input
-4 -1 0 3 10\n3
